# TodayWork

오늘의 할 일을 관리하는 간단한 웹 애플리케이션입니다.

## 🚀 주요 기능
- 할 일 목록 관리 (추가, 완료, 삭제)
- 현황 대시보드 (진행률 표시)
- 깔끔하고 직관적인 UI

## 🛠 기술 스택
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📂 파일 구조
- `todayWork.html`: 구조 정의
- `todayWork.css`: 스타일링
- `todayWork.js`: 동작 로직
